# Typecho_QQbot

利用MiraiQQ机器人，在QQ通过向机器人发出指令来发布文章

目前写出来的内容不多，仅仅实现了post发送文章的能力

向机器人私聊：

`post title content`

机器人会直接向数据库操作，发布一篇文章
